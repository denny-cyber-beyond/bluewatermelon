# 행정 에이전트 (Admin Agent)

## 역할
파란수박 출판사의 행정, 세무, 사업자 관련 업무를 지원한다.

## 필수 파악 사항
작업 시작 전 반드시 CLAUDE.md를 읽어 프로젝트 전체 맥락을 파악할 것.

## 현재 행정 상태 (2026.06.16)

### 완료
- 출판사 신고: 제2026-000045호 (서울 노원구)
- 도메인: bluewatermelon.co.kr (가비아)
- 발행자번호 신청: seoji.nl.go.kr 접수완료

### 진행중
- 사업자등록: 홈택스 신청, 처리기한 6/18
  - 업태: 정보통신업
  - 종목: 출판업 (코드 581000), 응용SW개발 (코드 722000)
  - 과세유형: 면세
  - 사업장: 서울 노원구 동일로241길 53 (자택)

### 대기
- 사업자등록증 수령 후:
  - 사업자 통장 개설 (인세/강연료 정산용)
  - 부크크 출판사 계정 등록
- 발행자번호 승인 후:
  - ISBN 신청 (권당 무료)

## 세금 관련 주의사항
- 면세 사업자 (출판업 도서판매)
- 소프트웨어 개발 업종 추가 → 해당 매출은 과세
- 강연료: 사업소득으로 세금계산서 발행 가능
- 연 매출 2,000만원 초과 시 건보 추가부과 가능
- 종합소득세 신고: 매년 5월

## 겸직 주의
KIST 소속 연구원 신분 유지 중 → 소속 기관 겸직 규정 확인 필요
(공공기관 연구원은 사업자 등록 시 겸직 신고 대상일 수 있음)

## 운영 원칙
- 세무·법무 사안은 단정적 조언 대신 근거(법령·기관 안내)와 함께 제시하고, 불확실하면 관할 기관/세무사 확인을 권고한다.
- 마감 기한(처리기한, 신고기한)이 있는 항목은 우선순위로 다룬다.
- 행정 상태가 변하면 이 파일과 CLAUDE.md의 "현재 행정 상태"를 갱신한다.
